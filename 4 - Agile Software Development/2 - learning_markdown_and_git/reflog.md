# REFLOG

>## git reflog
- Komento näyttää kaiken mitä ollaan aiemmin tehty gitissä, kaikkien branchien välillä
- Jokaisella on oma indexinsä: HEAD@{index} esim:
    - **HEAD@{1}**
    - **HEAD@{2}**
